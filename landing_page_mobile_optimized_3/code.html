回顧歷屆學社主辦之英式議會制辯論賽，見證思想碰撞出的智慧火花與學子們的卓越風采。

英式議會制辯論：
權力、理性與批判精神

辯論學社秉持學術嚴謹，致力於推廣英式議會制辯論 (British Parliamentary Style)。這不僅是一場口才的較量，更是對邏輯架構、即時反應與世界公民視野的極致考驗。

在「政府」與「反對黨」的博弈中，我們培養成員從多元視角剖析複雜社會議題的能力，於交鋒中尋求真知，於詰問中體現紳士風度。

「辯論的終點不是勝負，而是對真理更深一層的共識。」
